  ```
  git init                        - creates a git repo out of the directory it is in
  git status                      - show status

  git add file.text               - add files to staging area
  git commit -m "message here"    - commit file to local repo with message
  git diff                        - check file difference

  git log

  git clone https://url-to-clone  - clone repo into my working directory

  git push origin master          - push from default remote repository to master


  ```
  in this order
```
git diff
git status
git add
git commit
- then type in the message, once saved, then it will be committed
git log - for verifying
```
